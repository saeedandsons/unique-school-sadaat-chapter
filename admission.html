Admission Enquiry

Fill the form below and click "Send via WhatsApp" — admission details will be prepared and opened in WhatsApp with the phone number you provided.

Tip: If you are on desktop and WhatsApp Desktop/web isn't installed, clicking the button will open WhatsApp Web in a new tab.

Phone used for WhatsApp message will be auto-converted: if you enter a local Pakistani number starting with 0 (like 0325...), it will be sent to WhatsApp as international format (prefix 92) automatically. To use another country code, enter number with country code (e.g. +1XXXXXXXXXX).
